Registration Data Access Protocol (RDAP) is the modern replacement for WHOIS, developed by the IETF. It provides:

Privacy Protection

Standardized redaction of personal information

Internationalization

Supports UTF-8 and non-ASCII characters

JSON Format

Machine-readable responses for automation

RESTful API

Standardized endpoints and HTTP methods

Feature RDAP WHOIS
